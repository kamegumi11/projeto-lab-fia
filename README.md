# Home Credit Default Risk

## Objetivo

Prever a probabilidade de inadimplência de clientes utilizando
Machine Learning e Deep Learning.

## Dataset

Competição Kaggle Home Credit Default Risk.

## Tecnologias

- Python
- Pandas
- Scikit-Learn
- XGBoost
- LightGBM
- TensorFlow
- SHAP

## Arquitetura

Raw → Processed → Features → Model → Predictions

## Modelos

- Logistic Regression
- Random Forest
- XGBoost
- LightGBM
- Neural Network (MLP)

## Métrica

ROC-AUC

## Resultados

| Modelo | ROC AUC |
|----------|---------|
| Logistic Regression | 0.72 |
| Random Forest | 0.74 |
| XGBoost | 0.78 |
| MLP | 0.76 |
